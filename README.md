# FinClub Open Project 2

## Objective
Predict missing implied volatility values in the options dataset.

## Methodology
1. Data preprocessing and strike extraction
2. Separate CE and PE modelling
3. Volatility curve fitting
4. Convexity-adjusted interpolation/extrapolation
5. Temporal smoothing using historical information

## Files
- iv_surface_prediction.ipynb
- submission.csv
- filled dataset.csv

## Reproducibility
Running the notebook from start to finish reproduces the submitted CSV file.
